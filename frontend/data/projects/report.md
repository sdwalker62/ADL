This file will describe the structure and contents of the testing artifacts generated by Anvil.

## Basics

Once the testing pipelines are complete they will generate a report in the form of a JSON file. This file will be used build PDFs for customers as well as be read by METAL to generate the Forge Dashboard. Each model will have its own report file containing multiple runs.
